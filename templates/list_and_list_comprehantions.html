List and List Comprehantions

List is very inportent datatype in all programming language. If you are a C++ coader you may be familiar in Vector. If you are in java then Arraylist ot List.

In python we will delear a list as 

    mylist = []

This is an empty list.

We can define a list including many elements. 

    songs_list = ['folksong' , 'banglasong' , 'englishsong']

We can now delete the last element form songs_list List by this

songs_list.pop()

We can now delete the last element form songs_list List by its index

songs_list.pop(1)

This will delete the 2nd element form songs_list . So it delete 'banglasong' from the list

Now this is most powerful things you can do by using list comprehantion 


     num_list = [ 3 , 7 , 7 , 6, 1]
     [ number* number for  number in num_list ]

The output of this code is 

     [9, 49, 49, 36, 1]

Now Explanation about this code number* number this makes square of a number for number in num_list This will makes squres for all number in the num_list

Now this is most powerful things you can do by using list comprehantion 


     num_list = [ 3 , 7 , 7 , 6, 1]
